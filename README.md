# Student Management System

A **React** application for managing students with global state management using **Context API**. This project demonstrates prop drilling removal, global state sharing, combining local & global state, and custom hook usage for a scalable architecture.

---

## Features

- Add, view, and delete students.
- Search students by name.
- Filter students by major.
- Persistent storage using **localStorage**.
- Global state management using **React ContextAPI**.
- Input validation with **React Toastify** notifications.
- Responsive UI with clean component structure.
- 404 Page for unmatched routes.
- Custom hooks for better code reuse:
  - `useFetch` – fetch random users or external data.
  - `useForm` – manage form state and validation efficiently.
